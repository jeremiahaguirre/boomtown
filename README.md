## Server Instruction

Commands must be run from the `server` directory:

### Installation

Install the npm node modules like:

```bash
npm install
```

## Usage

### Run this:

```bash
npm run start:dev
```

### Test this:

- Just linting:

```bash
npm run lint
```

- Run linting, and fix any errors:

```bash
npm run lint:fix
```

- Run Jest tests:

```
npm run jest
```

- Run Jest tests, and watch for changes:

```bash
npm run jest:watch
```

- Run all tests:

```bash
npm run test
```

### Technologies Used

- Node
- Express
- Apollo
